What is a client?
The user. It could be a computer, tablet, or mobile device that is connected to the web.
What is a server?
It is a device that waits for requests from the user.
What is an HTTP request?
It is the query that is sent to the server. It contains the Type of request, the Header, and sometimes the Body.
What is an HTTP response?
It is the response sent back to the client. It contains the Status Code, the Header, and sometimes the Body.
What is Django and why do we use it?
It is Python modules that allow you to intercept requests and form responses.
